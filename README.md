# lofen

Lofen is a modern feature-packed terminal music player for local libraries, built around fast navigation and offline listening.
It is based on the modern UI of [dhonus/jellyfin-tui](https://github.com/dhonus/jellyfin-tui).

## Features

- local music library scanning from folders you choose
- offline playback by default
- custom themes, accent colors, and smooth visual transitions
- double-queue playback flow with shuffle, repeat, and queue controls
- playlists, search, sorting, and multi-library style organization
- embedded metadata and cover extraction from your audio files
- vim-style keybindings and SSH/tmux-friendly usage
